# BrighTv-Analysis Objective
This repository contains datasets and analytical resources designed to support Bright TV’s strategic objective of growing its subscription base during the current financial year. The focus is on enabling the Customer Value Management (CVM) team to better understand customer behavior  and make recommandations.

My role is to apply Sequal Query Language (SQL), aggregated data, and visualization skills to increase subscriber acquisition.

# 2. OBJECTION

Using the given dataset, I am providing solutions to the following key questions:

- Provide insights on user and usage trends of BrightTV.
- What type of factors influence consumption?
- What content would you recommend to increase consumption on the days with low consumption?
- What type of initiatives would you recommend to further grow BrightTV ‘s User base.
